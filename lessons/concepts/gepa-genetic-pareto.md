# GEPA (Genetic-Pareto)

## Term
**GEPA**: Genetic-Pareto

## Definition
Evolutionary LLM prompt optimization agent using genetic algorithms combined with Pareto selection principles

## Method
- **Genetic Search**: Evolutionary approach to prompt optimization
- **Pareto Selection**: Multi-objective optimization maintaining Pareto frontier
- **Natural-Language Reflection**: Uses LLM feedback to guide evolution

## Performance
- Outperforms GRPO and MIPROv2 by ~10-20%

- Maintains Pareto frontier across multiple objectives

Efficient prompt optimization for complex multi-objective scenarios where traditional single-objective optimization fails
