
### CI Failure Investigation
```bash
# Check which CI checks are failing
gh pr checks 1234 --repo gptme/gptme-contrib

# Get details on failures
gh pr checks 1234 --repo gptme/gptme-contrib --json name,state,link \
  --jq '.[] | select(.state == "FAILURE")'

# For each failing check, read the logs:
# gh run view RUN_ID --repo gptme/gptme-contrib --log-failed | tail -40
```

Also check master branch CI health:
```bash
gh run list --repo gptme/gptme-contrib --branch master --limit 3 --json name,conclusion,createdAt,url \
  --jq '.[] | select(.conclusion == "failure") | {name, conclusion, createdAt, url}'
```
